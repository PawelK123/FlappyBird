# 🐤 Flappy Bird (Python)

A clone of the popular Flappy Bird game built using Python and Object-Oriented Programming (OOP) with the Pygame library.

## 📖 Description

This project is a simple recreation of the classic Flappy Bird game. The player controls a bird and tries to avoid obstacles while achieving the highest possible score.

Before starting the game, the player can enter their name. After losing, the score is saved to a JSON file and displayed in the leaderboard.

## 🎮 Features

* 🐤 Classic Flappy Bird gameplay
* 🎮 Simple graphical interface using Pygame
* 🏁 Start menu with options:

  * Start game
  * View leaderboard
  * Exit game
* 🧾 Player name input before the game
* ⚡ Increasing difficulty — the game speeds up as the score increases
* 🏆 Leaderboard system (scores saved to JSON file)
* ⚙️ Object-Oriented structure

## 🛠️ Technologies

* Python
* Pygame
* JSON (for data storage)
